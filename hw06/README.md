## 2. 选题简介

本视频围绕“具身智能”主题展开，介绍了具身智能的基本概念、与传统人工智能的本质区别，以及“感知-决策-行动”三大核心技术环节。视频结合家用扫地机器人、工业机械臂、达芬奇手术机器人、特斯拉Optimus人形机器人等典型案例，展示了具身智能在家庭、工业、医疗等领域的应用现状与发展前景。结尾部分对“人机协作”的未来方向进行了简要讨论。

## 5. 学习心得

通过观看“小v讲AI”视频号系列短片，我学习了科普短视频的选题角度、叙事结构和画面呈现技巧。在本人视频制作中，借鉴了“问题引入—核心讲解—总结展望”的三段式结构，并结合具身智能主题进行内容组织。具体学习心得已另文提交。

## 6. 总结与反思

本次作业让我学会了如何将抽象技术概念通俗化表达，掌握了短视频制作的完整流程（脚本→录制→剪辑→导出），以及剪映的基本操作技巧。过程中遇到的困难包括概念理解不深入、素材分散、录制卡顿、音画不同步、时长不足等，通过查阅资料、分段录制、音频对齐、补充内容等方式逐一解决。今后将在画面丰富度、背景音乐控制、字幕校对、出镜表现力等方面继续改进。

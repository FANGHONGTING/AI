\# 📷 简单面部识别应用



本项目是一个基于 `Streamlit` 和 `face\_recognition` 构建的人脸检测与识别 Web 应用。用户上传图像后，系统会自动检测人脸位置，并可选地进行身份识别。



\---



\## 📁 项目结构



\- `src/app.py` : 主程序代码（含 Streamlit 界面 + 人脸处理逻辑）

\- `requirements.txt` : Python 依赖包列表

\- `README.md` : 本说明文档



\---



\## ⚙️ 功能说明



\### 人脸检测

1\. 用户上传图片。

2\. 系统自动检测图中所有人脸。

3\. 在原图上用红色方框标出人脸位置，并显示坐标。



\### 人脸识别（可选扩展）

> 若实现识别功能，流程如下：

1\. 预先准备“已知人脸库”（见下文）。

2\. 提取上传图像中人脸的特征编码。

3\. 与已知库比对，输出最匹配的身份或标记为 "Unknown"。



\---



\## 👥 如何准备人脸库（若实现识别）



1\. 在项目根目录创建文件夹 `known\_faces/`。

2\. 将已知人物的照片放入该文件夹。

3\. 文件名即为姓名（例如：`zhangsan.jpg`, `lisi.png`）。

4\. 程序启动时会自动加载这些照片作为参考库。



> 💡 示例：

> ```

> known\_faces/

> ├── zhangsan.jpg

> └── lisi.png

> ```
